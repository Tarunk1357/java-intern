## Objective
Create a simple HTML-based authentication system with 5 pages and submit the code via GitHub repository.
## Requirements
Create the following 5 HTML pages with basic redirections:
### Page 1: Login Page (login.html)
1. Username input field
• Password input field
• Login button (redirects to Dashboard)
• "Forgot Password?" link (redirects to Forgot Password page)
• "Create a New Account" link (redirects to Registration page)
### Page 2: Registration Page (register.html)
• Name input field
• Email ID input field
• Phone Number input field
• Password input field
• Confirm Password input field
• Register button (redirects to Login page)
• "I already have an account" link (redirects to Login page)
### Page 3: Forgot Password Page (forgot-password.html)
• Email input field
• "Send Password Reset Link" button (redirects to Login page)
Page 4: Reset Password Page (reset-password.html)
• New Password input field
• Confirm Password input field
• "Update Password" button (redirects to Login page)
### Page 5: Dashboard Page (dashboard.html)
• Display "Welcome to My Dashboard" heading
• Logout button (redirects to Login page)
